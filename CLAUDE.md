# Coweta Tote Rentals — project notes for Claude

This is the marketing website for **Coweta Tote Rentals**, a tote/dolly rental side business
run by **Justin and Chelsea Wade** (Coweta County, GA area). It is a static HTML site that
**deploys to Netlify automatically when changes are pushed to GitHub** (`main` branch).

When you open this project — locally or in a cloud session — read this file first, then work
in the voice and rules below.

## What this site is

- Plain static site: `index.html` plus `privacy.html`, `terms.html`, `accessibility.html`,
  `robots.txt`, `sitemap.xml`, and an `images/` folder. No build step, no framework.
- To make a change: edit the HTML directly. On push to `main`, Netlify redeploys.

## Voice rules (honesty over polish — they're new and don't want to mislead)

- **No em dashes or en dashes. Regular hyphens (-) only.**
- **Do not invent specifics** — no made-up call counts, referral numbers, or metrics. If a number
  or claim isn't confirmed, leave it out or ask.
- **Soften uncommitted things** to "we're testing / considering X" (course, expansion, insurance, FB group).
- **Don't recommend things they haven't actually done** (e.g. paid ads, aggressive inventory buys).
- **Don't oversell** the husband's future course — the goal is just to collect emails.
- No hero images of Chelsea; no "credit appreciated" disclaimers on shared assets.
- Pricing comparison points that ARE public/fair to cite: Stacks Online (charges ~3x), Home Depot's
  $100 moving-box package.

## When unsure

Chelsea would rather you **ask than guess.** If a change needs a fact you don't have
(a price, a claim, a detail about how they operate), ask instead of filling it in.
